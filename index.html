Welcome at my Side

A responsive website created by me.

This file exists, all right. In fact, I wrote it for the world to see. If you've [installed keybase] you can see this file at `/keybase/public/daniel156161/index.html`.

And YES, it works. When I edit index.html on my own computer, these sites all instantly update:

- https://keybase.pub/daniel156161 - [my files inside a glossy, navigable site]
- https://daniel156161.keybase.pub - [this file rendered as HTML]

If you're testing the Keybase filesystem, this will all work for you, too.

#### How it works

- [keybase.pub] is just a dumb site that serves things straight out of `/keybase/public`.
- It's not privileged in any way.
